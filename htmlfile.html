Unit I: World Wide Web Concepts

Concepts & Definitions

These topics are conceptual, but here is how they relate to code structure:

  • Web Page: A single HTML document.
  • Website: A collection of web pages linked together.
  • Static Website: Content is fixed (HTML/CSS only). Content only changes when the code is edited.
  • Dynamic Website: Content changes based on user interaction or database (PHP, Node.js, Python).
  • Client-Side Scripting: Code runs in the browser (e.g., JavaScript).
  • Server-Side Scripting: Code runs on the server (e.g., PHP, JSP, ASP.NET).

Unit I: HTML Overview

1. Document Structure & Comments

The basic building block of any HTML page.

<!-- This is an HTML Comment -->
<!DOCTYPE html>
<html>
    <head>
        <title>Page Title</title>
    </head>
    <body>
        Content goes here...
    </body>
</html>

2. Text Formatting & Special Characters

<!-- Headings -->
<h1>Heading 1</h1> to <h6>Heading 6</h6>

<!-- Paragraphs & Breaks -->
<p>This is a paragraph.</p>
<br> <!-- Line break -->
<hr> <!-- Horizontal Rule -->

<!-- Formatting -->
<b>Bold Text</b> or <strong>Important Text</strong>
<i>Italic Text</i> or <em>Emphasized</em>
<u>Underlined</u>
<sup>Superscript (x2)</sup>
<sub>Subscript (H2O)</sub>

<!-- Special Characters (Entities) -->
Space: &nbsp;
Less than (<): &lt;
Greater than (>): &gt;
Ampersand (&): &amp;
Copyright: &copy;

3. Anchors, Images & Sound

<!-- Anchor Tag (Hyperlink) -->
<a href="https://google.com" target="_blank">Click Here</a>

<!-- Adding Images -->
<img src="image.jpg" alt="Description" width="100">

<!-- Adding Sound (Audio) -->
<audio controls>
    <source src="music.mp3" type="audio/mpeg">
    Your browser does not support the audio element.
</audio>

4. Lists & Tables

<!-- Unordered List (Bullets) -->
<ul type="circle">
    <li>Item A</li>
    <li>Item B</li>
</ul>

<!-- Ordered List (Numbers/Letters) -->
<ol type="I">
    <li>First</li>
    <li>Second</li>
</ol>

<!-- Definition List -->
<dl>
    <dt>HTML</dt>
    <dd>Hypertext Markup Language</dd>
</dl>

<!-- Tables -->
<table border="1">
    <thead>
        <tr> <th>Name</th> <th>Age</th> </tr>
    </thead>
    <tbody>
        <tr> <td>John</td> <td>25</td> </tr>
    </tbody>
</table>

5. Frames & Floating Frames

Note: The <frame> and <frameset> tags are obsolete. The syllabus likely refers to Iframes (Floating Frames).
<!-- Floating Frame (iFrame) -->
<iframe src="https://www.wikipedia.org" width="300" height="200">
</iframe>

6. Developing Forms

<form action="/submit" method="POST">
    <!-- Text Input -->
    <label>Name:</label>
    <input type="text" name="username" placeholder="Enter name">

    <!-- Radio Button -->
    <input type="radio" name="gender" value="male"> Male

    <!-- Checkbox -->
    <input type="checkbox" name="subscribe"> Subscribe

    <!-- Dropdown List -->
    <select name="city">
        <option value="delhi">Delhi</option>
        <option value="mumbai">Mumbai</option>
    </select>

    <!-- Textarea -->
    <textarea rows="4" cols="50"></textarea>

    <!-- Submit Button -->
    <input type="submit" value="Send">
</form>

7. Image Maps

Making specific areas of an image clickable.

<img src="planets.jpg" usemap="#planetmap">

<map name="planetmap">
    <!-- Rectangular area -->
    <area shape="rect" coords="34,44,270,350" href="computer.htm">
    <!-- Circular area -->
    <area shape="circle" coords="337,300,44" href="coffee.htm">
</map>

Unit II: Cascading Style Sheets (CSS)

1. Types of Style Sheets (Syntax)

/* 1. Inline CSS (Inside the tag) */
<p style="color: red; font-size: 20px;">Hello</p>

/* 2. Internal CSS (Inside <head>) */
<style>
    p { color: blue; }
</style>

/* 3. External CSS (Linking a separate file) */
<link rel="stylesheet" type="text/css" href="styles.css">

2. CSS Properties, Styling, and Selectors

<style>
    /* Tag Selector */
    h1 {
        color: green;         /* Property: Value */
        text-align: center;
    }

    /* ID Selector (Unique, uses #) */
    #header {
        background-color: #f1f1f1;
    }

    /* Class Selector (Multiple use, uses .) */
    .error-msg {
        color: red;
        font-weight: bold;
    }

    /* Pseudo-Class (State of an element) */
    a:hover {
        text-decoration: underline;
        color: orange;
    }
</style>

Unit II: Bootstrap Basics

Important: Your syllabus asks for "Glyphicons" and "Jumbotron". These are primarily Bootstrap 3 components (Jumbotron existed in 4, but Glyphicons were removed in 4). The syntax below matches the specific keywords in your syllabus.

1. Linking Bootstrap

<!-- In <head> section -->
<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.4.1/css/bootstrap.min.css">
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
<script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.4.1/js/bootstrap.min.js"></script>

2. Container & Grids

<div class="container"> <!-- Fixed width container -->
    <div class="row">
        <!-- Grid system (Total 12 columns) -->
        <div class="col-sm-4">Column 1 (Span 4)</div>
        <div class="col-sm-4">Column 2 (Span 4)</div>
        <div class="col-sm-4">Column 3 (Span 4)</div>
    </div>
</div>

3. Components (Tables, Images, Buttons, Jumbotron)

<!-- Jumbotron (Big announcement box) -->
<div class="jumbotron">
    <h1>Welcome!</h1>
    <p>This is a jumbotron.</p>
</div>

<!-- Bootstrap Table -->
<table class="table table-striped table-bordered">...</table>

<!-- Responsive Image -->
<img src="pic.jpg" class="img-responsive img-rounded">

<!-- Buttons -->
<button class="btn btn-primary">Primary Button</button>
<button class="btn btn-success">Success Button</button>

<!-- Typography Classes -->
<p class="text-center">Centered Text</p>
<p class="text-uppercase">Uppercase Text</p>
<p class="text-danger">Red Text</p>

4. Glyphicons

Icons provided by Bootstrap (Version 3 specifically).

<p>Search Icon: <span class="glyphicon glyphicon-search"></span></p>
<p>Print Icon: <span class="glyphicon glyphicon-print"></span></p>
<p>User Icon: <span class="glyphicon glyphicon-user"></span></p>
